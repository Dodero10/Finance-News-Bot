📊 PHƯƠNG PHÁP TÍNH TOÁN:
------------------------------
1. ACCURACY:
   - Công thức: (Số câu gọi tools đúng hoàn toàn như ground truth) / Tổng số câu
   - Ý nghĩa: Tỉ lệ agent gọi đúng tất cả tools cần thiết và không gọi thừa
   - Ground truth: Dựa trên synthetic_news.csv

2. F1 SCORE, PRECISION & RECALL:
   - Dựa hoàn toàn trên ground truth từ synthetic_news.csv
   - Texp: Danh sách tools kỳ vọng (từ ground truth)
   - Tact: Danh sách tools agent trả về (loại bỏ failed tools)
   - TP: |Texp ∩ Tact| - Tools được gọi đúng và cần thiết
   - FP: |Tact - Texp| - Tools được gọi nhưng không cần thiết (thừa)
   - FN: |Texp - Tact| - Tools cần thiết nhưng không được gọi (thiếu)
   - Precision = TP / (TP + FP) = |Texp ∩ Tact| / |Tact|
   - Recall = TP / (TP + FN) = |Texp ∩ Tact| / |Texp|
   - F1 = 2 * (Precision * Recall) / (Precision + Recall)

3. TOOL FAIL RATE:
   - Công thức: (Số câu có failed_tools_count > 0) / Tổng số câu có gọi tools
   - Ý nghĩa: Tỉ lệ lỗi khi agent thực thi tools
