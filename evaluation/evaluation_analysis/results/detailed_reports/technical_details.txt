📊 PHƯƠNG PHÁP TÍNH TOÁN:
------------------------------
1. ACCURACY:
   - Công thức: (Số câu có failed_tools_count = 0) / Tổng số câu
   - Ý nghĩa: Tỉ lệ agent gọi tools hoàn toàn đúng, không có lỗi

2. F1 SCORE:
   - Dựa trên ground truth từ synthetic_news.csv
   - TP: Tools được gọi đúng và cần thiết
   - FP: Tools được gọi nhưng không cần thiết (thừa)
   - FN: Tools cần thiết nhưng không được gọi (thiếu)
   - Precision = TP / (TP + FP)
   - Recall = TP / (TP + FN)
   - F1 = 2 * (Precision * Recall) / (Precision + Recall)

3. TOOL FAIL RATE:
   - Công thức: (Số câu có failed_tools_count > 0) / Tổng số câu có gọi tools
   - Ý nghĩa: Tỉ lệ lỗi khi agent thực thi tools
